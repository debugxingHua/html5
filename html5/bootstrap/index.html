1、媒体查询

媒体查询有两个部分，一个设备规范，一个大小规则。
媒体查询允许设定最小最大的显示规则，可基于视口大小移动、显示或隐藏内容。
/* Bootstrap 在超小设备中，手机、也就是小于 768px 没有媒体查询 */
/* Bootstrap 在小型设备中，平板、也就是大于 768px @media(min-width:@screen-sm-min{...}) */
/* Bootstrap 在中型设备中，台式、也就是大于 992px @media(min-width:@screen-md-min{...}) */
/* Bootstrap 在大型设备中，大台式、也就是大于 1200px @media(min-width:@screen-lg-min{...}) */
也可以包含max-width： @media(min-width:@screen-md-min) and (max-width:@screen-md-max){...};

2、容器container、row、栅格化的12列

container是一个容器，保证了获得适当的对齐与内边距。而row呢，表示每个行，行可以嵌套。 栅格化的12列，处于一个row行中，可嵌套自己定义。

3、列偏移、列排序

列偏移col-md-offset-1;
列排序：
将列向左：col-md-pull-1;
将列向右：col-md-push-1;
